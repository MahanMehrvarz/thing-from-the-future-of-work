# Thing From The Future Of Work

A digital card deck application for envisioning future work scenarios.

## Features

- **Interactive Card Deck:** Flip through cards in categories: Arc, Terrain, Object, and Mood.
- **Master Card:** Each category has a "Master Card" (the first card with a star) that flips all cards in that category simultaneously.
- **Mix Mode:** "Make Futures" tab shuffles a random hand of cards.
- **Download:** Download individual cards or the entire high-resolution deck (dynamically generated).

## Development

### Install Dependencies
```bash
npm install
```

### Run Locally
```bash
npm run dev
```
